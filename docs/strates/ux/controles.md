# Contrôles clavier et souris

Synthèse des raccourcis disponibles dans l'éditeur Strates. L'overlay en jeu (touche **H**) reste la source de vérité visuelle, ce fichier sert de référence documentaire.

## Caméra

- **Z Q S D** (ou W A S D) : translation avant, gauche, arrière, droite.
- **A / E** : rotation azimut gauche, droite.
- **Clic gauche maintenu** (outil Naviguer) : pivoter la caméra.
- **Clic droit maintenu** : déplacer latéralement.
- **Molette** : zoom.

## Outils (raccourcis numériques et lettres)

| Touche | Outil |
| ------ | ----- |
| 1 | Naviguer (aucun ordre) |
| 2 | Miner un voxel ou un obstacle |
| 3 | Placer un bloc depuis les stocks |
| 7 | Poser une maison |
| 8 | Tracer un champ |
| 0 | Poser un laboratoire de recherche |
| P | Poser un promontoire d'observation (astronomie) |
| 4 | Poser une forêt (mode sandbox) |
| 5 | Poser un rocher (mode sandbox) |
| 6 | Poser un filon (mode sandbox) |
| 9 | Poser un buisson de baies (mode sandbox) |

## Actions contextuelles

- **Shift + clic gauche** : applique l'outil sur toute la strate (même biome, même hauteur) sous le curseur.
- **Shift + clic droit** : annule tous les ordres sur la strate survolée.
- **Shift + molette** (outil Filon) : change le type de filon posé.
- **Clic droit bref** sur un marqueur : annule l'ordre de cette tuile.
- **Clic gauche bref** sur un colon : ouvre sa fiche personnage.

## Cycle jour / nuit

- **N** : bascule jour / nuit (ou clic sur l'icône soleil/lune en haut).
- La nuit, les colons IDLE convergent vers le foyer le plus proche (feu de camp social).
- Un colon stationné sur un promontoire la nuit génère des points nocturnes (+1 toutes les 5 s).
- La cueillette de baies est désactivée la nuit (activité jour).

## Menus et méta

- **T** : arbre des technologies.
- **H** : overlay d'aide (contrôles).
- **?** : guide texte détaillé.
- **Échap** : menu pause, ou fermer un panneau ouvert.
- **R** : nouvelle partie (confirmation demandée).
